Red Blood Cell Detection



This project focused on the development of a smartphone-based system for detecting and classifying white blood cells (WBC) from microscopic images. The research was conducted to address limitations of manual cell analysis, which is time-consuming, prone to human error, and less practical for high-throughput medical environments. By leveraging image processing and pattern recognition techniques, the system aimed to provide a faster, more accessible, and cost-effective diagnostic support tool, particularly for regions with limited medical facilities.

The proposed system utilized the template matching method as the core detection approach due to its simplicity, flexibility, and low computational requirements. Unlike deep learningâ€“based object detection methods that require intensive training and high computational resources, template matching allows object detection by comparing image patterns with predefined templates. This characteristic made it highly suitable for deployment on smartphone devices, enabling efficient processing without relying on high-end hardware.

Microscopic image datasets of white blood cells were obtained from publicly available online sources and categorized into four cell types: neutrophils, lymphocytes, monocytes, and eosinophils. The system processed input images captured or selected via the smartphone camera, applied template matching to localize candidate cells, and displayed bounding boxes around detected regions. Performance optimization was conducted by evaluating different numbers of templates and threshold values to balance detection accuracy and computational efficiency.

Experimental results demonstrated that the system achieved an average processing time of approximately 264.67 milliseconds per image, making it suitable for real-time or near real-time mobile applications. An optimal threshold value of 0.6 was identified, resulting in a detection accuracy of 85% while maintaining efficient computation. Additionally, using four templates was found to be the most effective configuration, offering the best trade-off between accuracy and processing speed without exceeding mobile device limitations.

Overall, this project highlights the practical integration of computer vision, image analysis, and mobile computing for medical applications. The system demonstrates how lightweight algorithms such as template matching can be effectively utilized on resource-constrained devices to support medical diagnostics, improve accessibility, and reduce dependency on expensive laboratory equipment.
